Aula: Introdução ao GitHub

Construa seu site e aprenda a compartilhar com o mundo!

O que é o GitHub?

O GitHub é um lugar na internet onde você pode guardar seus projetos de programação. É como um caderno onde você escreve suas receitas, mas no mundo da tecnologia.

É super legal porque outras pessoas podem ver o que você fez, te ajudar a melhorar e até usar o que você criou. É como compartilhar suas receitas com amigos para todos cozinharem juntos!

Criando um Repositório no GitHub

Repositório é apenas uma palavra chique para uma pasta especial para o seu projeto. É nesse lugar que você vai guardar todos os arquivos do seu site, como fotos e textos.

  1. Vá para o GitHub e crie uma conta ou faça login.
  2. Clique no "+" no canto superior direito e escolha "Novo repositório".
  3. Dê um nome para o seu repositório, algo como meusite.
  4. Escolha se quer que todos vejam seu projeto (público) ou só você (privado).
  5. Clique em "Criar repositório".

Seu repositório é como uma pasta vazia pronta para você colocar suas receitas (ou códigos, no caso!).

Criando um Site no GitHub

Para fazer seu site, você precisa de uma página inicial, certo? Chame essa página de index.html e coloque tudo nela.

Depois, é só acessar o seu site em: https://seunome.github.io/meusite, trocando seunome pelo seu nome de usuário e meusite pelo nome do repositório que você criou.

Agora todo mundo pode ver o que você cozinhou, quer dizer, programou!

Organizando seu Projeto

Quando você cozinha, gosta de ter tudo organizado, certo? Com o código do seu site é a mesma coisa!

  • Crie pastas para colocar diferentes coisas, como fotos, músicas e vídeos.
  • Dê nomes que façam sentido para suas pastas e arquivos. É como organizar seus brinquedos por categorias.
  • Não esqueça de explicar o que cada coisa faz. É como seguir uma receita para não se perder na cozinha!
